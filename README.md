# Diplom_Yandex
Автоматизация теста к API
Теперь автоматизируй сценарий, который подготовили коллеги-тестировщики:
Клиент создает заказ.
Проверяется, что по треку заказа можно получить данные о заказе.
Шаги автотеста:
Выполнить запрос на создание заказа.
Сохранить номер трека заказа.
Выполнить запрос на получение заказа по треку заказа.
Проверить, что код ответа равен 200.

Необходимое ПО:
Python 3.8+
pip (для установки зависимостей)
pytest (для запуска тестов)

Файлы проекта
order_test.py — основной файл с тестами.
sender_stand_request.py — модуль для отправки HTTP‑запросов.
configuration.py — константы (URL, пути, заголовки).

Запуск всех тестов выполянется командой pytest order_test.py -s
Перед запуском тестов обнови URL_SERVICE в configurtion.py без "/" в конце
